# Landing de librojuegos (GitHub Pages)

Esta carpeta contiene una landing responsive (PC/tablet/móvil) para:
- Descargas por librojuego (Fichas / Misiones-Expansiones / Material extra)
- Bloques plegables (para que no sea interminable)
- Botón "Comprar en Amazon" + QR (placeholder) por librojuego

## Publicación rápida en GitHub Pages
1) Crea un repositorio llamado `librojuegos`
2) Sube el contenido de esta carpeta al repositorio
3) Settings → Pages → Deploy from a branch → main / (root)

## Sustituir portadas
Reemplaza los archivos `img/*-cover.jpg` manteniendo el mismo nombre.

## Sustituir QRs
Reemplaza `img/*-qr.png` por el QR real.

## Subir PDFs
Sube PDFs a:
- downloads/SLUG/fichas/
- downloads/SLUG/misiones/
- downloads/SLUG/material-extra/

Mantén nombres para no romper enlaces compartidos.
