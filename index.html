Cómo actualizar (sin programación)

Para publicar cambios solo tendrás que subir archivos y, cuando quieras, copiar una línea para añadir un enlace nuevo.

1) Sustituir una portada

Reemplaza el archivo de imagen en la carpeta img/ manteniendo el mismo nombre (por ejemplo: la-llave-del-abismo-cover.jpg). No necesitas tocar nada más.

2) Añadir un PDF nuevo (misión, expansión o extra)

Sube el PDF a la carpeta correspondiente dentro de downloads/SLUG/: fichas, misiones o material-extra. Luego añade una línea de enlace en el bloque del librojuego (yo lo dejaré señalado en el HTML).

3) Enlace y QR de Amazon

Cambia el enlace del botón Comprar en Amazon por el URL real (o el ASIN). Sustituye el QR en img/SLUG-qr.png. El botón abre Amazon en una pestaña nueva para que el usuario no pierda la landing.
